# SpellFire 
##### By Kevin Naranjo and Will Rusk


## Controls
Player Movement - *W, A, S, D*

Camera Movement - *Mouse (__Left click to toggle camera movement__)*

Debug Hitbox Toggle - *M*

Quit Game - *Escape Key*
